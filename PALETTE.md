# Future Buddy Color Palettes

## Palette 1 - Primary

| Name | Hex | Used For |
|------|-----|----------|
| Intense Cyan | `#00F0FF` | Icon background |
| Deep Purple | `#8800FF` | Icon color |
| Gold | `#FFD700` | Icon stroke |
| Purple-Black | `#100820` | Icon frame |
| Yellow-Green | `#ADFF2F` | Badge |
| Deep Forest | `#3D5C00` | Badge stroke |
| Dark Purple | `#2A0055` | Shadow |

### Design Notes
- Icon uses abstract shape from game-icons.net
- Gold + Cyan glow used for demo video captions
- Yellow-Green badge intentionally "messes with your eyes" against gold
- Purple-Black frame lets neon colors pop

### CSS Variables
```css
:root {
  --fb-cyan: #00F0FF;
  --fb-purple: #8800FF;
  --fb-gold: #FFD700;
  --fb-frame: #100820;
  --fb-badge: #ADFF2F;
  --fb-badge-stroke: #3D5C00;
  --fb-shadow: #2A0055;
}
```
