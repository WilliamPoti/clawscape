# Recordings

## Structure

- `horizontal/` - 16:9 format videos
- `shorts/` - 9:16 format videos
- `scripts/` - Written voiceover scripts
- `voiceovers/` - Generated audio files

## Naming

Rename files from `futurebuddy-*` to `0000XXX demoname.mp4`

Example: `0000002 multiplayer.mp4`

**Always rename before moving on to the next demo.**

## YouTube

Title format: `Future Buddy #0000XXX` (match file name)

Example: `Future Buddy #0000002`

Privacy: Unlisted

See `CONTENT_PIPELINE.md` in project root for TTS setup.
